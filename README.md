# Scoop Bucket for dotsider

[![Tests](https://github.com/willibrandon/scoop-bucket/actions/workflows/ci.yml/badge.svg)](https://github.com/willibrandon/scoop-bucket/actions/workflows/ci.yml) [![Excavator](https://github.com/willibrandon/scoop-bucket/actions/workflows/excavator.yml/badge.svg)](https://github.com/willibrandon/scoop-bucket/actions/workflows/excavator.yml)

[Scoop](https://scoop.sh) bucket for [dotsider](https://github.com/willibrandon/dotsider) — a TUI for analyzing .NET assemblies.

## Usage

```powershell
scoop bucket add dotsider https://github.com/willibrandon/scoop-bucket
scoop install dotsider
```

## Updating

```powershell
scoop update dotsider
```

## Available Packages

| Package | Description |
|---------|-------------|
| [dotsider](https://github.com/willibrandon/dotsider) | A TUI for analyzing .NET assemblies |
